学习前端第1天——两列定宽结构

两列定宽结构一般就是一个左侧边(.mainBox)、一个右侧边(.sideBox)。我会把这两个放进一个#container的容器里。

接下来HTML结构如下：

<div id="header">头部信息</div>

<div id="container">

<div class="mainBox">

主要内容区域<br/>

主要内容区域<br/>

主要内容区域<br/>

</div>

<div class="sideBox">次要内容区域</div>

</div>

<div id="footer">底部信息</div>

现在我们给上CSS样式填充，在浏览器进行浏览看下样子

CSS样式如下：

*{

margin: 0;

padding: 0;

}

#header, #footer{

width: 960px;

height: 30px;

background-color: #E8E8E8;

}

#container{

height: 250px;

width: 960px;

margin: 10px 0;

}

.mainBox{

height: 250px;

width: 680px;

float: left;

background-color: #333333;

}

.sideBox{

height: 250px;

width: 270px;

float: right;

background-color: #333333;

}

浏览器浏览如下：

 

这里有个问题，当.mainBox左侧边随内容高度不断填充的时，内容高度如果高过容器高度，内容会溢出容器。当然我们可以利用CSS属性overflow解决这个问题，将多余的部分隐藏或设置滚动条。但这里想设置的容器会随着内容的填充高度自增。

想容器的高度随内容高度自增，我们就要先删除#container,.mainBox,.sideBox三个的高度，接下来浏览器浏览：

 

很明显的底部信息位置变了，我们这里要添加清除浮动来设置，接下来CSS样式：

*{

margin: 0;

padding: 0;

}

#header, #footer{

width: 960px;

height: 30px;

background-color: #E8E8E8;

}

#container{

/*height: 250px;*/

width: 960px;

margin: 10px 0;

}

.mainBox{

/*height: 250px;*/

width: 680px;

float: left;

background-color: #333333;

}

.sideBox{

height: 250px;

width: 270px;

float: right;

background-color: #333333;

}

#footer{

clear: both;

}

浏览器浏览：

 

接着我们又看到内容区域(#container)与底部信息(#footer)很明显的下边距(margin)没有了，使用下面CSS样式可以进行修改:

#footer{

clear:both;

} /* 清除底部信息的左右浮动 */

#container:after{

display: block;

clear: both;

content: "";

}
